# Guia para enviar fotos, textos y sugerencias

Este archivo sirve para pedir informacion a personas que no van a editar la pagina directamente.

La idea es simple:

- La persona revisa la pagina.
- Reune textos, fotos y comentarios.
- Te entrega todo en una sola carpeta o archivo.
- Luego esos cambios se revisan y se suben al sitio.

## 1. Que debe entregar la persona

Puede enviar uno o varios de estos elementos:

- Fotos nuevas.
- Logos.
- Textos corregidos.
- Datos de contacto actualizados.
- Observaciones sobre errores o mejoras.
- Documentos institucionales autorizados.

## 2. Como debe organizar la entrega

Lo ideal es que entregue una carpeta con este contenido:

```txt
aportes-nombre-persona-fecha/
```

Ejemplo:

```txt
aportes-hernando-sandoval-2026-05-26/
```

Dentro de esa carpeta puede incluir:

- una carpeta `fotos/`
- una carpeta `logos/`
- una carpeta `documentos/`
- un archivo de texto con comentarios

## 3. Archivo de comentarios

Debe incluir un archivo de texto o Word con observaciones claras.

Nombre sugerido:

```txt
comentarios.txt
```

o

```txt
comentarios.docx
```

Formato recomendado para cada observacion:

```txt
Pagina:
Seccion:
Cambio sugerido:
Texto nuevo:
Motivo:
Prioridad: alta | media | baja
```

Ejemplo:

```txt
Pagina: Inicio
Seccion: Hero principal
Cambio sugerido: Ajustar titulo principal
Texto nuevo: Diseño, fabricación y montaje de equipos para agroindustria y manejo de graneles.
Motivo: Explica mejor lo que hace la empresa.
Prioridad: alta
```

## 4. Si va a enviar fotos

Cada foto debe tener un nombre claro.

Evitar:

- `foto1.jpg`
- `imagen nueva.png`
- `final final 2.jpg`

Usar mejor:

- `tolva-portuaria-buenaventura.jpg`
- `grua-montaje-meta.jpg`
- `equipo-proyecto-orf-pore.jpg`

Ademas, en el archivo de comentarios debe indicar:

```txt
Nombre del archivo:
Que muestra:
En que pagina deberia aparecer:
Proyecto o cliente relacionado:
Ciudad o ubicacion:
Observacion adicional:
```

## 5. Si va a enviar textos

No sirve mandar frases sueltas sin contexto.

La persona debe indicar:

- en que pagina va el texto
- en que seccion va
- cual es el texto nuevo completo

Eso evita interpretaciones y retrabajo.

## 6. Si va a enviar logos

Debe indicar:

- nombre de la empresa
- si ese logo si esta autorizado para publicarse
- si reemplaza uno actual o si es nuevo

## 7. Si va a enviar documentos

Solo debe enviar documentos que puedan publicarse.

Tambien debe indicar:

- nombre oficial del documento
- fecha
- version
- si reemplaza otro documento existente

## 8. Lo que conviene pedirle a la persona

Puedes copiarle este texto:

```txt
Por favor envíame una sola carpeta con tus aportes.
Incluye allí las fotos, textos o documentos nuevos y un archivo de comentarios explicando qué cambiar, en qué parte de la página y por qué.
Si mandas imágenes, ponles nombres claros y dime para qué sección son.
Si mandas textos, envíalos completos y no como notas sueltas.
```

## 9. Que no conviene recibir

- Fotos sin nombre claro.
- Textos por WhatsApp sin contexto.
- Archivos duplicados.
- Documentos sin confirmar si son públicos.
- Comentarios vagos como "mejorar diseño" sin indicar dónde.

## 10. Recomendacion final

La mejor entrega no es la más grande sino la más clara.

Si la persona manda todo mezclado, luego se pierde tiempo separando archivos, entendiendo qué quería cambiar y adivinando dónde iba cada cosa.
